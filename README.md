# Hadiwijaya | AI Consulting & Solutions

Welcome to the official web platform for Hadiwijaya, a specialized AI consulting firm. 

We empower businesses by providing:
- **AI Consulting**: Expert guidance on optimizing workflows with Artificial Intelligence.
- **Custom ML Models**: Tailor-made machine learning architectures designed for specific enterprise needs.
- **AI Playground**: An interactive chat interface to test conversational AI models directly within our platform using a BYOK (Bring Your Own Key) model.
- **Secure Integrations**: Utilizing verified Google Auth platform standards for seamless and secure OAuth integrations.

## Tech Stack
This project is built using:
- [Next.js](https://nextjs.org/) (Pages Router)
- React
- Vanilla CSS / CSS Modules
- Vercel deployments

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to view the application locally.
